# prd-conversational: 交互式 PRD 构建器

## 功能

多轮引导对话 → 逐模块确认 → 完整 PRD。

## 使用方法

说"开始对话式 PRD"激活本技能。

## 对话流程

| 阶段 | 问题数 | 内容 |
|------|--------|------|
| 阶段0 | 1个 | 意图理解 |
| 阶段1 | 4个 | 平台与基础 |
| 阶段2 | 5个 | 核心功能 |
| 阶段3 | 2个 | 技术偏好 |
| 阶段4 | 3个 | 质量要求 |

## 对比方案A

| | 方案A（全自动） | 方案B（交互式） |
|--|-------------|--------------|
| 用户参与 | 低 | 高 |
| PRD 准确性 | 依赖推断 | 用户确认 |
| 问题数量 | 0个 | 15个 |
| 耗时 | 秒级 | 分钟级 |
